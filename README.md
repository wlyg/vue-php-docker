- 1.安装docker，docker-compose
- 2.进入项目根目录
    - ./start.sh init 初始化，拉取容器镜像
    - ./start.sh up 启动容器
    - ./start.sh stop 停止容器
    - ./start.sh build 进入容器编译代码
        - backend: composer (composer.json), -->vender
           - composer install
        - frontend: npm (package.json), -->node_modules
           - cnpm install
           - npm run build<env>  -->dist
           - npm run buildLocal  --> .env.local
